# 🧱 Arkanoid Clone - Rebota, rompe y programa tu primer juego retro

¡Bienvenido/a al proyecto *Arkanoid Clone*! 🕹️  
Este proyecto es un homenaje al clásico de los 80s, pero creado desde cero por ti usando **Unity**.  
Aprenderás programación, diseño de mecánicas y trabajo colaborativo con **Git y GitHub Desktop**. 🧠💥

---

## 🎯 Objetivos del proyecto

✅ Asentar los fundamentos de **programación orientada a objetos**  
✅ Entender las físicas básicas en Unity (colisiones, rebotes, velocidad)  
✅ Familiarizarse con **prefabs, scripts y escenas**  
✅ Aprender a usar **Git y GitHub Desktop** para el trabajo en grupo  
✅ Recrear un clásico con toques personales y mejoras jugables

---

## 🧠 ¿Qué aprenderás?

🔧 Programación en **C#** desde lo más básico  
🎮 Desarrollo de un videojuego en **Unity**, paso a paso  
📁 Organización de recursos (sprites, sonidos, scripts)  
🌍 Uso práctico de **repositorios Git y GitHub**  
🔁 Trabajo por versiones, colaboración y resolución de conflictos

---

## 🧰 Tecnologías utilizadas

- 🎮 **Unity** (versión recomendada: 2022.3 o superior)
- 💻 **C#**
- 🌐 **Git y GitHub Desktop**
- 🔊 Sonidos retro y recursos visuales personalizados o de uso libre

---

## 🚀 ¿Cómo comenzar?

1. Abre GitHub Desktop y clona este repositorio.
2. Abre la carpeta del proyecto con Unity.
3. Ejecuta la escena principal y… ¡rompe ladrillos sin parar!
